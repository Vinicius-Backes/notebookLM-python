# Resumo Estruturado do Assunto

## 1. Introdução ao Python

Python é uma linguagem de programação de alto nível, conhecida pela sintaxe simples e facilidade de aprendizado.

## 2. Variáveis e Tipos de Dados

Foram estudados tipos como:

* inteiros (`int`)
* reais (`float`)
* texto (`str`)
* booleanos (`bool`)

## 3. Estruturas Condicionais

Uso de:

* `if`
* `elif`
* `else`

para controle de decisões no programa.

## 4. Estruturas de Repetição

Utilização de:

* `for`
* `while`

para automatizar tarefas repetitivas.

## 5. Funções

Criação de funções para reutilização e organização do código.

## 6. Estruturas de Dados

Estudo de:

* listas
* tuplas
* dicionários

## 7. Tratamento de Erros

Uso de `try` e `except` para evitar falhas inesperadas.

## 8. Aplicações do Python

Foram exploradas aplicações em:

* automação
* desenvolvimento web
* análise de dados
* inteligência artificial

# Glossário

| Termo       | Definição                                  |
| ----------- | ------------------------------------------ |
| Variável    | Espaço utilizado para armazenar valores    |
| Função      | Bloco de código reutilizável               |
| Lista       | Estrutura que armazena múltiplos elementos |
| Loop        | Estrutura de repetição                     |
| Dicionário  | Estrutura de chave e valor                 |
| Condicional | Estrutura usada para tomada de decisão     |
| Exceção     | Erro tratado durante a execução            |
| Biblioteca  | Conjunto de funcionalidades prontas        |
